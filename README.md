6. Herramientas y tecnologías
● Frontend: Angular 20, TypeScript, RxJS, HTML5, CSS3, Angular Material.
● Backend: Java 21, Spring Boot 3.x, Spring Data JPA, Spring Security, Maven.
● Base de datos: MySQL, PostgreSQL o MongoDB.
● Control de versiones: Git + GitHub (entrega obligatoria con commits regulares).
● Entorno de despliegue: XAMPP / Tomcat / Docker.
7. Entregables del proyecto
1. Documento de análisis y diseño (diagrama ER, casos de uso, arquitectura).
2. Código fuente completo (Angular + Spring) con control de versiones en GitHub.
3. Manual técnico de instalación y uso.
4. Vídeo de defensa (10 min) mostrando funcionalidad y roles del grupo.
5. Informe individual con:
○ Funcionalidad desarrollada por el alumno.
○ Dificultades técnicas encontradas.
○ Propuesta de mejora o refactorización.
8. Evaluación individual y seguimiento
Estrategias de evaluación individual:
1. Repositorio personal en GitHub Classroom (cada alumno realiza commits con su
usuario).
2. Revisiones periódicas (sprints semanales o quincenales con entregables
parciales).
3. Cuaderno de trabajo individual: cada alumno documenta su aportación en el
informe.
4. Defensa oral individual (10 min) sobre su módulo o componente.
5. Evaluación cruzada del grupo (coevaluación): cada alumno valora el trabajo de
los compañeros.

9. Extensiones opcionales
● Implementar WebSockets (Spring + Angular) para comunicación en tiempo real.
● Añadir modo offline con almacenamiento local (IndexedDB / Service Workers).
● Implementar API REST externa (por ejemplo, integración con Google Contacts o
Twilio).
● Añadir módulo de informes PDF con generación automática desde el servidor.
10. Conclusión
Este proyecto intermodular integra todos los pilares del desarrollo de software profesional:
● Diseño visual y experiencia de usuario (DI).
● Gestión de datos y persistencia (AD).
● Automatización y lógica empresarial (SGE).
● Servicios concurrentes y asincronía (PSP).
Cada alumno será responsable de una parte crítica del sistema, lo que garantiza una
evaluación individual justa y una visión completa del ciclo de desarrollo.
